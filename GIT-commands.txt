GIT commands
git clone 
git status 

git add - команда гіт, каже гіту що цей файл потрібно відслідковувати. Додаємо підверсійний контроль.

git add . / -А - add all files

git commit - m "короткий опис коміту, те що ми додали через git add"

git push - відправляємо дані у репозиторій гіт

git pull - отримуємо дані з репозиторія
